Questão 1

Enunciado: Utilize o método da substituição para calcular a complexidade de tempo de pior caso do algoritmo recursivo abaixo. No algoritmo abaixo, o parâmetro V da função é um vetor de inteiros e o parâmetro N é o tamanho do vetor. A operação de divisão usada é a divisão inteira, ou seja, é a divisão que sempre resulta em um inteiro.

Algoritmo: Algoritmo(V, n)

Entrada: um vetor V com n inteiros

Saída: soma dos elementos do vetor V

  1. Se n == 0 entao
  2.     Retorne 0
  3. Senao
  4.     soma = 0
  5.     Para j = 1, ..., n faça
  6.         soma = soma + V[j]
  7.     Retorne soma + Algoritmo(V, n/2)
  1. Forneça uma hipótese para T(n) e prove por indução que ela é válida.

    2. Resposta:

    Linha Tempo de Execução
    1 \( \Theta(1) \)
    2 \( \Theta(1) \)
    3 \( \Theta(1) \)
    4 \( \Theta(1) \)
    5 \( \sum_{j=1}^{n} (1) = n\)
    6 \( (\sum_{j=1}^{n} (1)) - 1 = (n - 1)\)
    7 \( T(\frac{n}{2}) + \Theta(1) \)

    Análise de Complexidade:

    Vamos calcular a complexidade \( T(n) \) com base na soma de todas as linhas da tabela

    Como funciona essa soma?

    1. Se \( n = 0 \), \( T(n) = \Theta(1) = 1 \);
    2. Se \( n > 0 \), \( T(n) = T(\frac{n}{2}) + n + (n - 1) + \Theta(1) = T(\frac{n}{2}) + 2n \)

    Agora vamos provar que essa recorrência é válida, usando indução:

    Palpite

    Vou palpitar que essa recorrência \( T(n) = O(n) \)

    Hipótese de Indução

    Seja \(k\), tal que \( n_0 \leq k \leq n \), então:

    $$ T(k) \leq ck $$

    Passo indutivo

    $$ T(n) = T(\frac{n}{2}) + 2n $$

    Pela hipótese de indução, temos:

    $$ T(n) \leq \frac{cn}{2} + 2n $$

    Agora, vamos comparar \( T(n) \), com o limite que queremos provar, para encontrar um valor de \( c \), que torne a inequação verdadeira.

    $$ \frac{cn}{2} + 2n \leq cn $$

    $$ (\frac{c}{2} + 2)n \leq cn $$

    $$ \frac{c}{2} + 2 \leq c $$

    $$ \frac{c + 4}{2} \leq c $$

    $$ 2c\leq c + 4 $$

    $$ c \geq 4 $$

    Assim, podemos afirmar que para valores de \( c \geq 4 \), a inequação é verdade.

    Portanto, \( T(n) = \Theta(n) \)

Questão 2

Enunciado: É possível provar que a solução para a recorrência \(T(n)=4T(n/3)+n\) é \(T(n)=\Theta(n^{\log_3 4})\). Mostre que uma prova de substituição que considere \(T(n)\leq cn^{\log_3 4}\) falha. Depois, mostre como subtrair um termo de ordem mais baixa para fazer com que uma prova de substituição funcione. Basta provar apenas o limitante superior, não precisa provar o inferior.

  1. Mostre que a prova com \(T(n)\leq cn^{\log_3 4}\) falha.

    2. Resposta:

    Vamos tentar provar que a recorrência é válida, por indução:

    Seja \(k\), tal que \( n_0 \leq k \leq n \), então:

    $$ T(k) \leq ck^{\log_3 4} $$

    Agora, vamos considerar \( T(n) \)

    $$ T(n)=4T(\frac{n}{3})+n $$

    $$ T(n)=4c \frac{n^(log_3 4)}{3^(log_3 4)} + n $$

    $$ T(n)=4c \frac{n^(log_3 4)}{4} + n $$

    $$ T(n)= cn^(log_3 4) + n $$

    Com o limite que palpitamos, não é possivel continuar com a prova.

  2. Mostre como subtrair um termo de ordem mais baixa para fazer a prova funcionar.

    3. Resposta:

    Agora, vamos fortalecer nossa hipótese

    Seja \(k\), tal que \( n_0 \leq k \leq n \), e \( a > 0 \) então:

    $$ T(k) \leq ck^{\log_3 4} - ak $$

    Agora, vamos considerar, \( T(n) \)

    $$ T(n)=4T(\frac{n}{3})+n $$

    Por Hipótese de indução, temos:

    $$ 4 ( c (\frac{n^(\log_3 4)}{3^(\log_3 4)}) - a \frac{n}{3} ) + n $$

    $$ (\frac{4cn^(\log_3 4)}{4}) - 4a \frac{n}{3} + n $$

    $$ cn^(\log_3 4) - \frac{4an}{3} + n $$

    $$ cn^(\log_3 4) - n(\frac{4a}{3} - 1) $$

    Para que isso seja menor que \( cn^(\log_3 4) - an\), precisamos que:

    $$ \frac{4a}{3} - 1 \leq a $$

    $$ \frac{4a - 3}{3} \leq a $$

    $$ 4a - 3 \leq 3a $$

    $$ a \geq 3 $$

    Com isso, podemos afirmar que \( T(n) \) é \( O(n^(\log_3 4) - an ) \), para valores de \(a \geq 3\)

Questão 4

Enunciado: Use o teorema mestre para fornecer limites assintóticos restritos para as recorrências a seguir.

  1. \(T(n) = 2T(n/4) + 1\)

    2. Resposta:

    \(a = 2, b = 4, f(n) = 1\)

    \(n^{\log_4 2} = n^{1/2}\)

    \(f(n) = 1 \leq cn^{1/2 - \varepsilon}\) para \(\varepsilon = 1/2\), e \( c = 1 \)

    Portanto, pelo caso 1 do teorema mestre: \(T(n) = \Theta(n^{2/4})\)

  2. \(T(n) = 2T(n/4) + n^2\)

    3. Resposta:

    \(a = 2, b = 4, f(n) = n^2\)

    \(n^{\log_4 2} = n^{1/2}\)

    \( f(n) = n^2 \geq cn^{1/2 + \varepsilon} \), para \( \varepsilon = 1/2\) e \( c = 1 \)

    Assim, temos que \( f(n) = \Omega(n) \)

    Para encontramos o limite assintótico, usando o caso 3 do teorema mestre, temos que provar ainda que: \( af( \frac{n}{b} ) \leq cf(n)\)

    \( 2f( \frac{n}{4} ) \leq cf(n)\)

    Vamos assumir que \( c = \frac{1}{2} \)

    \( 2 (\frac{n}{4})^2 \leq \frac{n^2}{2} \)

    \( \frac{2n^2}{16} \leq \frac{n^2}{2} \)

    \( \frac{n^2}{8} \leq \frac{n^2}{2} \)

    Podemos afirmar que a inequação é verdade, pois o \( n^2 \) do lado esquerdo está sendo dividido por um numero maior que o \( n^2 \) do lado direito, tornando-se sempre menor, para valores de \( n \geq 1 \)

    Portanto, pelo caso 3 do teorema mestre: \(T(n) = \Theta(n^2)\)

  3. \(T(n) = 2T(n/4) + n\)

    4. Resposta:

    \(a = 2, b = 4, f(n) = n\)

    \(n^{\log_4 2} = n^{1/2}\)

    \(f(n) = n \geq n^{\frac{1}{2} + \varepsilon} \) para \(\varepsilon = 1/2\)

    Assim, temos que \( f(n) = \Omega(n) \)

    Para encontramos o limite assintótico, usando o caso 3 do teorema mestre, temos que provar ainda que: \( af( \frac{n}{b} ) \leq cf(n)\)

    \( 2f( \frac{n}{4} ) \leq cf(n)\)

    \( 2\frac{n}{4} \leq cn\)

    Para \( c = \frac{1}{2} \)

    \( \frac{2n}{4} \leq \frac{n}{2} \)

    \( \frac{n}{2} \leq \frac{n}{2} \)

    Para valores de \( n \geq 1 \), é possivel afirmar a corretude da inequação.

    Portanto, pelo caso 3 do teorema mestre: \(T(n) = \Theta(n^2)\)

  4. \(T(n) = 2T(n/4) + \sqrt{n}\)

    5. Resposta:

    \(a = 2, b = 4, f(n) = \sqrt{n} \)

    Vamos usar o caso 2 do teorema mestre, para encontrar os limites assintóticos dessa recorrência.

    Primeiro vamos provar que \( f(n) = O(n^{\log_4 2}) \)

    \(n^{\log_4 2} = n^{1/2}\)

    \(f(n) = \sqrt{n} \leq n^{1/2} \)

    \(f(n) = \sqrt{n} \leq \sqrt{n} \)

    Agora vamos provar que \( f(n) = \Omega(n^{\log_4 2}) \)

    \(f(n) = \sqrt{n} \leq n^{1/2} \)

    \(f(n) = \sqrt{n} \leq \sqrt{n} \)

    Com isso, é possivel afirmar que \( f(n) = \Theta(n^{2/4}) \)

    Portanto, pelo caso 2, do teorema mestre: \(T(n) = \Theta(n^{2/4} \log n)\)
